# AI Cloud Log Analyzer

A simple Flask-based web app that uses AI to analyze cloud logs, detect potential issues, and suggest fixes.

- Built with Python + Flask
- Supports OpenAI API (optional) or mock responses
- Highlights severity of issues
- Shows estimated cost of AI analysis

## Demo

- Run locally: `python app.py` → open `http://127.0.0.1:5000/`
- Load example logs → click **Analyze Logs** → see structured analysis
